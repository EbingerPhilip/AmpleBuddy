You might constantly get problems from the imports from "react-router-dom". That's a lib that allows React to have multiple pages without reloading the browser everytime.
For it, do:
npm install react-router-dom
npm install -D @types/react-router-dom

Now to run this entire thang, open the frontend in the terminal, and go:
npm run build

That should create a dist/ directory with the index.html in it.
Then you can start the server and see if it works (it should !).

For the JWT, you need to do:
npm install jsonwebtoken
npm install -D @types/jsonwebtoken

For the sidebar, I'm using react-icons, so:
npm install react-icons

// https://vite.dev/config/

    proxy: {
        "/api" : {
            target: "http://localhost:3000",
            changeOrigin: true,
        }

  }
})
